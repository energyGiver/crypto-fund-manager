# Crypto Tax Calculator - Frontend

Modern dark SaaS UI for crypto tax calculation.

## Quick Start

```bash
npm install
npm run dev
```

Frontend runs on `http://localhost:3001`

## Environment

```env
NEXT_PUBLIC_API_URL=http://localhost:3000
```

## Pages

- `/` - Landing page with address input
- `/loading/[jobId]` - Progress tracking
- `/report/[jobId]` - Tax report results
